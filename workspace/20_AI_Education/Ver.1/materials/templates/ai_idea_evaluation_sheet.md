## 評価軸

### 1. 効果（Effect）

| アイデアNo | 時間削減 | 品質向上 | リスク低減 | 効果スコア（合計） |
|-----------|---------|---------|-----------|------------------|
| 1 | 1-5 | 1-5 | 1-5 |  |
| 2 |  |  |  |  |
| 3 |  |  |  |  |
| 4 |  |  |  |  |
| 5 |  |  |  |  |

**スコアリング基準:**

- **時間削減 (1-5)**
  - 5: 50%以上削減
  - 4: 30-49%削減
  - 3: 10-29%削減
  - 2: 10%未満削減
  - 1: 効果なし

- **品質向上 (1-5)**
  - 5: エラー率90%以上削減
  - 4: エラー率50-89%削減
  - 3: エラー率10-49%削減
  - 2: エラー率10%未満削減
  - 1: 効果なし

- **リスク低減 (1-5)**
  - 5: 重大リスク（法規制・情報漏洩）を防止
  - 4: 中程度リスク（顧客クレーム）を大幅削減
  - 3: 小規模リスクを削減
  - 2: リスク削減効果が限定的
  - 1: 効果なし

### 2. 実現性（Feasibility）

| アイデアNo | データ可用性 | システム連携難度 | コスト | 実現性スコア（平均） |
|-----------|------------|----------------|--------|-------------------|
| 1 | 1-5 | 1-5 | 1-5 |  |
| 2 |  |  |  |  |
| 3 |  |  |  |  |
| 4 |  |  |  |  |
| 5 |  |  |  |  |

**スコアリング基準:**

- **データ可用性 (1-5)**
  - 5: 必要データが整備済み、即座に利用可能
  - 4: データは存在、軽微な整形で利用可能
  - 3: データ収集に1-2週間必要
  - 2: データ収集に1ヶ月以上必要
  - 1: データが存在しない、取得困難

- **システム連携難度 (1-5、逆スコア)**
  - 5: 連携不要、単独で完結
  - 4: 既存APIで簡単に連携可能
  - 3: カスタム連携が必要だが実績あり
  - 2: 複雑な連携が必要、実績なし
  - 1: 技術的に実現困難

- **コスト (1-5、逆スコア)**
  - 5: 無料ツールで実現可能
  - 4: 月額1万円以内
  - 3: 月額1-5万円
  - 2: 月額5-10万円
  - 1: 月額10万円以上または初期投資100万円以上

| 3 |  |  |  |  |  | High/Med/Low |
| 4 |  |  |  |  |  | High/Med/Low |
| 5 |  |  |  |  |  | High/Med/Low |

**総合スコア計算式:**

```
総合スコア = (効果スコア × 2) + 実現性スコア + 体験スコア
```

※ 効果スコアを2倍にして重み付け

**優先度判定基準:**

- **High**: 総合スコア 40以上
- **Medium**: 総合スコア 25-39
- **Low**: 総合スコア 24以下
