## 使い方

このテンプレートは、AI活用プロジェクトのROI（投資対効果）を測定し、工数削減・品質向上・リスク低減の効果を可視化するためのツールです。

### 使用ステップ
1. 事前データ（Before）を収集
2. AI導入後データ（After）を収集
3. 効果を定量化
4. ROIを計算
5. Session13の成果報告に活用

## 5. ROI計算

### 初年度

| 項目 | 金額（円） |
|------|----------|
| **効果合計** | **2,784,000** |
| - 工数削減効果 | 414,000 |
| - 品質向上効果 | 270,000 |
| - リスク低減効果 | 2,100,000 |
| **コスト合計** | **610,000** |
| - 初期投資 | 610,000 |
| **純効果（初年度）** | **2,174,000** |
| **ROI（初年度）** | **356%** |

```
ROI = (効果合計 - 初期投資) / 初期投資 × 100
    = (2,784,000 - 610,000) / 610,000 × 100
    = 356%
```

---

### 2年目以降（年間）

| 項目 | 金額（円/年） |
|------|-------------|
| **効果合計** | **2,784,000** |
| **コスト合計** | **560,000** |
| **純効果（2年目以降）** | **2,224,000** |
| **ROI（2年目以降）** | **397%** |

```
ROI = (効果合計 - 年間運用コスト) / 年間運用コスト × 100
    = (2,784,000 - 560,000) / 560,000 × 100
    = 397%
```
