# プロンプトとは？

> AIに対する「指示文」や「入力文」のこと

**良いプロンプト** → **良い応答**
**曖昧なプロンプト** → **曖昧な応答**

AIは「読心術」ができません。
**明確に伝えることが重要です。**

---

# プロンプトの4要素

```
┌─────────────────────────────────────────┐
│                                         │
│   ① 指示（Instruction）                │
│      └ 何をしてほしいか                 │
│                                         │
│   ② 文脈（Context）                    │
│      └ 背景情報・状況                   │
│                                         │
│   ③ 制約（Constraints）                │
│      └ 条件・制限                       │
│                                         │
│   ④ 出力形式（Output Format）          │
│      └ どんな形で返してほしいか         │
│                                         │
└─────────────────────────────────────────┘
```

---

# ① 指示（Instruction）

**AIに何をしてほしいかを明確に伝える**

### 良い例
- 「〇〇について説明してください」
- 「〇〇のメリットとデメリットを列挙してください」
- 「〇〇を〇〇形式で作成してください」

### 悪い例
- 「〇〇について」（動詞がない）
- 「〇〇どう思う？」（曖昧）

---

# ② 文脈（Context）

**背景情報を与えて、AIの理解を助ける**

### 例
```
私は中小企業の人事担当者です。
新卒採用の説明会で使用するプレゼン資料を作成しています。
当社は製造業で、従業員50名程度です。
```

**文脈があることで、AIは状況に合った回答ができます。**

---

# ③ 制約（Constraints）

**条件や制限を設けて、期待する範囲に絞る**

### 例
- 「300文字以内で」（長さ）
- 「専門用語を使わずに」（表現）
- 「敬語で」（トーン）
- 「〇〇については言及しないで」（禁止事項）

---

# ④ 出力形式（Output Format）

**どんな形式で返してほしいかを指定する**

### 例
- 「箇条書きで」
- 「表形式で」
- 「〇〇と△△の2つに分けて」
- 「まず結論、次に理由の順で」

---

# 4要素を組み合わせた例

```
【指示】新入社員向けのビジネスマナー研修の概要を作成してください。

【文脈】対象は4月入社の新卒社員20名です。
研修時間は2時間を予定しています。

【制約】専門用語は避け、具体例を多く含めてください。

【出力形式】以下の形式で出力してください：
- 研修タイトル
- 学習目標（3つ）
- タイムテーブル（表形式）
- 準備物リスト
```

---

# 4要素の覚え方

```
┌─────────────────────────────────────────┐
│                                         │
│         「し・ぶ・せ・だ」              │
│                                         │
│   し = 指示（何をしてほしい？）         │
│   ぶ = 文脈（どんな状況？）             │
│   せ = 制約（どんな条件？）             │
│   だ = 出だし形式（どう返す？）         │
│                                         │
└─────────────────────────────────────────┘
```

---

# 全部必要？

### 答え: 状況による

| 状況 | 必要な要素 |
|---|---|
| 簡単な質問 | 指示だけでOK |
| 業務利用 | 4要素すべて推奨 |
| 複雑なタスク | 4要素＋例示 |

**まずは4要素を意識する習慣をつける**

---

# 1-1 まとめ

### プロンプトの4要素

| 要素 | 役割 | キーワード |
|---|---|---|
| ① 指示 | 何をしてほしいか | 動詞で明確に |
| ② 文脈 | 背景情報 | 状況を伝える |
| ③ 制約 | 条件・制限 | 範囲を絞る |
| ④ 出力形式 | 返答の形 | 形式を指定 |

**「し・ぶ・せ・だ」で覚える**
