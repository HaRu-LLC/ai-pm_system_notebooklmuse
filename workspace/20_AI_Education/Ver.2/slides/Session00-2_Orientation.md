# 1. 本プログラムで得られること

---

# プログラムのゴール

## 「AIを業務で使いこなし、組織に貢献できる人材になる」

### レベル2からレベル3へ

- プロンプトの設計ができる
- 自分専用のGPTsを構築できる
- 業務改善の提案ができる（ROI試算付き）
- 組織内でAI活用を推進できる

---

# 3タイプの受講者に対応

| タイプ | 現状 | 本プログラム後 |
|---|---|---|
| **A: 積極利用者** | 日常的にAI活用 | さらに高度な活用、組織推進役へ |
| **B: これから使う人** | 興味はあるが未活用 | 業務でAIを自在に活用 |
| **C: ゼロスタート** | ほぼ未経験 | 自信を持ってAIを使える |

**どのレベルからでも、確実にステップアップ**

---

# 戦略→戦術→実行を繋ぐ

```
┌─────────────────────────────────────────┐
│  戦略級: Why / What                     │
│  → 何のためにAIを使うか                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  戦術級: How                            │
│  → どうやって実現するか                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  オペレーション級: Do                   │
│  → 実際に使う・作る・改善する           │
└─────────────────────────────────────────┘
```

**本プログラムは3層すべてをカバー**

---

# 他の学習方法との違い

| 方法 | メリット | デメリット |
|---|---|---|
| YouTube/書籍 | 手軽、無料 | 体系化されていない、継続困難 |
| 社内研修 | 業務に近い | 短時間、基礎のみ |
| 資格試験対策 | 体系的 | 実践機会がない |
| **本プログラム** | **体系＋実践＋伴走** | 13週間のコミットが必要 |

# 2. 13週間のロードマップ

---

# Phase構成

```
┌─────────────────────────────────────────────────────────┐
│  Phase 1        Phase 2        Phase 3        Phase 4  │
│  Week 1-3       Week 4-5       Week 6-10      Week 11-13│
│                                                         │
│  プロンプト     AIとの共創     GPTs構築       業務改善   │
│  基礎                          知識ベース     ROI分析    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

# Phase 1: 手書きプロンプト基礎（Week 1-3）

### 学ぶこと
- プロンプトの4要素（指示・文脈・制約・出力形式）
- 文脈と制約の設計技法
- 出力形式の最適化

### 成果物
- **プロンプト改善ログ**（成長の軌跡）

---

# Phase 2: AIとの共創（Week 4-5）

### 学ぶこと
- 対話型プロンプト設計
- AIを壁打ちパートナーとして活用
- プロンプトのテンプレート化

### 成果物
- **共創プロンプト原案**（再利用可能なテンプレート）

---

# Phase 3: カスタム指示・知識ベース（Week 6-10）

### 学ぶこと
- GPTsの概念と設計
- Instructions設計・知識ベース構築
- テストと改善サイクル

### 成果物
- **GPTs仕様書＋知識ベース**（運用可能なGPTs）

---

# Phase 4: 業務GPTs実運用と最終発表（Week 11-13）

### 学ぶこと
- 業務プロセス分析（AI適用ポイント発見）
- 自社データでGPTs構築・実運用
- 20分の最終発表

### 成果物
- **業務GPTs＋運用ログ＋発表資料**（実際に使えるツール）

---

# 0-2 まとめ

### 本プログラムで得られること

- レベル2→3へのステップアップ
- 3タイプの受講者それぞれに対応
- 戦略・戦術・実行の3層をカバー

### 13週間のロードマップ

- Phase 1-4で段階的にスキルアップ
- 各Phaseで具体的な成果物を作成

**→ 次の動画（0-3）では、学習の進め方と修了要件を説明します**
