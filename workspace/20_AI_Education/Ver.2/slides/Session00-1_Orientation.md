# 1. 企業が直面している課題

---

# AI時代の企業課題

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  「ツールは導入した。でも使われていない」      │
│                                                 │
│  「一部の人は活用しているが、全社に広がらない」│
│                                                 │
│  「効果が出ているのか、測定できていない」      │
│                                                 │
└─────────────────────────────────────────────────┘
```

**→ 「ツール導入」だけでは成果は出ない**

---

# 組織が抱える4つの課題

| 課題 | 具体的な症状 |
|---|---|
| **属人化** | 社長の頭の中にしかノウハウがない |
| **人材不足** | DX/IT人材がいない。採用も困難 |
| **情報過多** | AIツールが多すぎて選べない |
| **実行力不足** | 戦略は描けても、現場に落とし込めない |

---

# よくある失敗パターン①

### 「全員にChatGPTを配布したが、使われない」

```
┌──────────────────────────────────────┐
│  症状                                │
│  ・社員全員にアカウント配布          │
│  ・ほとんど使われていない            │
│  ・「何に使えばいいかわからない」    │
└──────────────────────────────────────┘
```

**原因**: ゴール・ユースケースが不明確

---

# よくある失敗パターン②

### 「レベル2社員のストレス蓄積」

```
┌──────────────────────────────────────┐
│  症状                                │
│  ・やる気ある社員が                  │
│    「会社のAI環境はショボい」        │
│    と不満を感じている                │
│  ・能力を発揮できずに離職            │
└──────────────────────────────────────┘
```

**原因**: 適切な環境・成長機会がない

---

# AI活用レベルと現実

| レベル | 定義 | 全社員比率 |
|:---:|---|:---:|
| **3** | ツールを作り出せる人 | 5% |
| **2** | 積極的に使う人（日常業務でAI活用） | 15% |
| **1** | 使う人（促されれば使う） | 50% |
| **0** | 使わない人（興味なし/抵抗あり） | 30% |

**全員に同じ研修をしても効果は薄い**

---

# 効果的なアプローチ

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   全員にChatGPTを配布 ──→ ほとんど使われない           │
│   （よくある失敗）                                      │
│                                                         │
│   レベル2に集中投資し、レベル3に引き上げる             │
│        ↓                                               │
│   レベル3がツールを作り、レベル1も巻き込む             │
│        ↓                                               │
│   組織全体のAI活用レベルが底上げされる                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

# 2. なぜ「研修だけ」では解決しないのか

---

# 従来のサービスの限界

| サービス | 強み | 弱み |
|---|---|---|
| **大手コンサル** | ブランド、網羅的 | 高額、中小企業に不向き |
| **SIer** | 技術力、大規模開発 | 戦略支援なし、柔軟性低い |
| **AI研修会社** | 体系的カリキュラム | **研修だけで終わる。実務定着しない** |
| **SaaS/ツール** | 安価、手軽 | **使い方を教えてくれない** |

---

# 「ミッシングリンク」問題

```
┌───────────────────────────────────────────────────┐
│                                                   │
│   戦略   →→→ ??? →→→   戦術   →→→ ??? →→→   実行│
│                                                   │
│   コンサルが作った        SIerが作った           │
│   戦略レポート           システム                 │
│         ↓                     ↓                  │
│   現場に落とし込めない   使われない               │
│                                                   │
└───────────────────────────────────────────────────┘
```

**戦略と実行が繋がらない = 成果が出ない**

---

# 研修の限界

### AI研修を受けても...

- 研修は受けた。でも現場で使えない
- 知識は増えた。でも業務に適用できない
- 一時的に使った。でも続かない

**知識だけでは「使える人材」にはならない**

---

# 必要なのは「実務定着」

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   【従来】                                              │
│   研修受講 → 終了 → 現場に戻る → 忘れる               │
│                                                         │
│   【本プログラム】                                      │
│   動画学習 → 実習課題 → フィードバック → 業務で活用  │
│      ↑                                     ↓           │
│      └──────── 13週間の継続サイクル ─────────┘          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
