# 2. 修了要件と評価方法

---

# 修了要件

| 要件 | 基準 |
|---|---|
| 動画視聴率 | 80%以上（13セッション中11以上） |
| 必須課題提出率 | **100%**（13課題） |
| 最終発表 | 参加必須（Week 13） |
| ミーティング参加 | 5回以上（7回中） |

---

# 課題の評価方法

### 自動評価（Gemini API）

- 提出後、すぐに評価結果が届く
- スコア（0-100点）＋フィードバックコメント

### 評価軸

| 軸 | 配点 |
|---|:---:|
| 課題理解度 | 25点 |
| 実践性 | 25点 |
| 創意工夫 | 25点 |
| 完成度 | 25点 |

---

# 評価結果の活用

```
┌──────────────────────────────────────────┐
│                                          │
│  スコアは「成績」ではなく「学習指針」    │
│                                          │
│  ・自分の強み/弱みを把握                 │
│  ・改善ポイントを次の課題に活かす        │
│  ・修了判定には使用しない                │
│                                          │
└──────────────────────────────────────────┘
```

**提出すれば修了できる。スコアを気にしすぎない。**
