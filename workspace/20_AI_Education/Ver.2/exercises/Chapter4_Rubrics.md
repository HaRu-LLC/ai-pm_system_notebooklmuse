#### EX-21評価プロンプト

```
あなたは教育評価の専門家です。以下の「業務分析メタプロンプト＋フロー図」課題を評価してください。

## 評価対象
- 業務分析メタプロンプト
- 変数定義シート
- AI生成の業務フロー図＋ボトルネック分析＋期待改善効果
- 振り返り

## 評価基準（各20点、合計100点）

1. **メタプロンプト設計（20点）**
   - 業務情報（業務名・担当者・頻度・所要時間等）を入力変数として設計しているか
   - 出力形式（フロー図・ボトルネック分析・期待改善効果）が明確に指定されているか
   - 品質基準（DoD）が含まれているか

2. **フロー図の完成度（20点）**
   - 全作業ステップが時系列で記載されているか
   - 各ステップに所要時間・担当者が明示されているか
   - インプット・アウトプットが明確か

3. **ボトルネック分析（20点）**
   - 時間・品質・待ち時間の3類型で分析されているか
   - ボトルネックの根拠が具体的か
   - 影響度（高/中/低）が明示されているか

4. **期待改善効果（20点）**
   - 時間削減効果が定量的に記載されているか（%または分）
   - 品質改善効果が具体的に記載されているか
   - コスト削減効果が算出されているか

5. **振り返りの質（20点）**
   - 業務分析の学びを言語化しているか
   - Phase 4（最終課題）への活用方針があるか

## 出力形式
{
  "total_score": 0-100,
  "criteria_scores": {
    "metaprompt_design": 0-20,
    "flowchart_completeness": 0-20,
    "bottleneck_analysis": 0-20,
    "expected_improvement": 0-20,
    "reflection_quality": 0-20
  },
  "feedback": "具体的なフィードバックコメント",
  "strengths": ["良かった点1", "良かった点2"],
  "improvements": ["改善点1", "改善点2"]
}
```
