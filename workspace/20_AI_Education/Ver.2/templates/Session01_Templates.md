# Session 1 テンプレート集

AIポリテラシー育成プログラム Ver.2
Phase 1 / Week 1

---

## 使い方

1. 受講アプリからこのテンプレートを**ダウンロード**してください
2. 必要なテンプレートを選んで、〇〇の部分を自分の内容に置き換えてください
3. 完成したらChatGPTに入力して実行してください
4. 課題を受講アプリで提出してください

---

## テンプレート1: 4要素プロンプト（EX-01用）

```
【指示】
〇〇してください。

【文脈】
私は〇〇です。〇〇という状況で使用します。

【制約】
- 〇〇文字以内
- 〇〇なトーンで
- 〇〇は含めないでください

【出力形式】
以下の形式で出力してください：
-
-
-
```

### 記入例（自己紹介プロンプト）

```
【指示】
自己紹介文を作成してください。

【文脈】
私は営業職5年目です。転職活動中で、IT企業向けの面接で使用します。
これまでBtoB法人営業を担当し、新規開拓と既存顧客フォローの両方を経験してきました。

【制約】
- 1分で話せる長さ（約300文字）
- 熱意が伝わるトーンで
- 転職理由には言及しない

【出力形式】
話し言葉で、段落分けして出力してください。
```

## テンプレート2: Before/After比較（EX-02用）

```
## Before

【プロンプト】
（あなたの最初のプロンプト）

【問題点】
- 問題点1
- 問題点2

---

## After

【プロンプト】
（改善後のプロンプト）

【改善ポイント】
- 改善ポイント1（〇〇を追加したことで、△△になった）
- 改善ポイント2（〇〇を具体化したことで、△△になった）
```

### 記入例

```
## Before

【プロンプト】
メール書いて

【問題点】
- 内容が抽象的すぎて何のメールかわからない
- 誰から誰へのメールか不明
- トーンの指定がないため汎用的すぎる

---

## After

【プロンプト】
【指示】取引先へのお礼メールを作成してください。
【文脈】先日の商談で契約が決まりました。担当者は田中部長です。
【制約】丁寧な敬語、簡潔に（200文字程度）、感謝の気持ちを込めて
【出力形式】件名と本文を分けて

【改善ポイント】
- 【文脈】を追加したことで、具体的な状況に合ったメールになった
- 【制約】で文字数とトーンを指定したことで、適切な長さと丁寧さになった
- 【出力形式】で件名と本文を分けたことで、すぐにコピペできる形式になった
```

## よくあるプロンプト例

### メール作成

```
【指示】〇〇へのメールを作成してください。
【文脈】〇〇という状況です。相手は〇〇です。
【制約】〇〇なトーンで、〇〇文字程度
【出力形式】件名と本文を分けて
```

### 議事録作成

```
【指示】以下の会議メモから議事録を作成してください。
【文脈】〇〇部の〇〇会議（参加者〇名）
【制約】正式な議事録形式、箇条書きで簡潔に
【出力形式】
- 日時・参加者
- 議題
- 決定事項
- アクションアイテム（担当・期限付き）

【会議メモ】
（ここに生のメモを貼り付け）
```

### 企画書の素案

```
【指示】〇〇の企画書の素案を作成してください。
【文脈】〇〇向けの企画です。目的は〇〇です。
【制約】A4 1枚程度、専門用語を避ける
【出力形式】
- タイトル
- 目的・背景
- 概要
- スケジュール
- 予算概算
```

---

## 困ったときは

- **結果が長すぎる** → 「〇〇文字以内で」と制約を追加
- **専門的すぎる** → 「初心者にもわかるように」と追加
- **的外れな回答** → 文脈情報を詳しく書く
- **形式が違う** → 出力形式を明確に指定する
