# Session 11 テンプレート集

## 概要

Session 11「業務プロセス分析」の実習課題で使用するテンプレートです。

| 課題 | テンプレート | 用途 |
|---|---|---|
| EX-21 | 業務分析メタプロンプトテンプレート | 業務フロー図・ボトルネック分析を生成 |
| EX-22 | AI適用候補リスト生成テンプレート | 優先順位付きAI適用候補を生成 |

---

## EX-21: 業務分析メタプロンプト＋フロー図（必須）

### 課題の目的

自社業務の概要をメタプロンプトに入力し、AIに業務フロー図・ボトルネック分析を生成してもらう。Phase 4の最終課題に向けた業務選定の基盤となる。

### 提出物

1. 業務分析メタプロンプト（変数化されたもの）
2. 変数定義シート
3. AI生成の業務フロー図＋ボトルネック分析＋期待改善効果
4. 振り返り

### タイムボックス推奨

| 作業 | 推奨時間 | 上限 | 備考 |
|---|---|---|---|
| 対象業務の選定 | 10分 | 15分 | 自分が担当する業務を選ぶ |
| メタプロンプト設計 | 15分 | 20分 | テンプレートをベースに |
| メタプロンプト実行 | 15分 | 20分 | AIに分析させる |
| 生成結果レビュー・修正 | 15分 | 25分 | フロー図・ボトルネックを確認 |
| 振り返り記入 | 10分 | 15分 | |
| **合計** | **65分** | **95分** | |

> **ポイント**: Phase 4で構築するGPTsの対象業務を選定する。フロー図が正確でなくても、ボトルネックが見えればOK

### テンプレート4: 業務情報入力サンプル

```
## 1. 業務基本情報
業務名: 営業日報作成

1回あたりの所要時間: 約30分/人

## 2. 作業内容（時系列で記述）
1. 本日の活動履歴を思い出す
2. Excelテンプレートを開く
3. 訪問先・商談内容を入力する
4. 数値データ（商談金額等）を入力する
5. 翌日の予定を記入する

- 思い出しながら書くため時間がかかる

- 社内CRM（参照のみ）
```

## EX-22: AI適用候補リスト（任意）

### 課題の目的

EX-21で生成したフロー図・ボトルネック分析を入力として、AI適用候補の優先順位付きリストを生成する。GPTs化の方針を固める。

### 提出物

```

以下の業務フローとボトルネック分析に基づき、AI適用候補リストを生成してください。

{EX-21で生成したフロー図}

【入力: ボトルネック分析】
{EX-21で生成したボトルネック分析}

【分析観点】
1. 効果（時間削減インパクト）
   - 高: 50%以上の時間削減
   - 中: 20-50%の時間削減
   - 低: 20%未満の時間削減

2. 実現性（技術的ハードル）
   - 高: ChatGPT/GPTsで完結
   - 中: API連携が必要
   - 低: 専用システム開発が必要

3. 優先度マトリクス
   - ◎（優先実施）: 効果 高×実現性 高
   - ○（余裕あれば）: 効果 高×実現性 中、または 効果 中×実現性 高
   - △（要検討）: 効果 中×実現性 中
```
