# Session 2 テンプレート集

AIポリテラシー育成プログラム Ver.2
Phase 1 / Week 2

---

## 使い方

1. 受講アプリからこのテンプレートを**ダウンロード**してください
2. 必要なテンプレートを選んで、〇〇の部分を自分の内容に置き換えてください
3. 完成したらChatGPTに入力して実行してください
4. 課題を受講アプリで提出してください

### EX-03用 サンプル会議メモ

> 以下の会議メモをそのままコピーして使用してください。
> 3パターンすべてで同じ会議メモを使うことで、ペルソナ・制約の違いによる出力の変化を比較できます。

```
【会議メモ（サンプル）】

日付: 2026年1月6日（月）10:00-11:00
場所: 第1会議室
参加者: 田中部長、鈴木課長、佐藤、山田、高橋

■ 議題1: 新システム導入について
・来月から本格稼働開始で決定
・プロジェクトリーダーは田中さん、サブリーダーは鈴木さん
・予算は500万円で承認済み、追加が必要な場合は別途相談
・導入スケジュール：
  - 1月中: 要件定義
  - 2月: 開発・テスト
  - 3月: 本稼働
・ベンダーは株式会社テックソリューションに決定

■ 議題2: 今月の売上進捗
・目標達成率: 現時点で85%
・残り15%の見込み：
  - 新規案件A（山田担当）→ 来週契約予定、500万円
  - 既存顧客B（佐藤担当）→ 追加発注の可能性あり
・月末までに目標達成の見込み

■ 議題3: 来月のキャンペーン企画
・高橋さんが素案を作成中
・来週水曜までに部内共有
・予算申請は再来週の経営会議で

■ その他
・既存顧客Bからシステム不具合のクレームあり
  → 佐藤さんが対応中、今週中に解決予定
・田中部長が来週木曜は出張で不在

■ 次回
・来週月曜 10:00〜 同じ会議室
```

## テンプレート2: 業務プロンプト3本セット（EX-04用）

### メール作成プロンプト

```
【指示】〇〇への〇〇メールを作成してください。

【文脈】
私は〇〇部の〇〇です。
〇〇という状況で、〇〇のためにメールを送ります。
相手は〇〇（役職・関係性）です。

【制約】
- 〇〇文字程度
- 〇〇なトーンで（丁寧な敬語/ビジネスカジュアル等）
- 〇〇を必ず含める
- 〇〇には言及しない

【出力形式】
件名と本文を分けて出力してください。
```

### 記入例（お礼メール）

```
【指示】取引先への商談お礼メールを作成してください。

【文脈】
私は営業部の山田です。
昨日、株式会社ABC様と新規案件の初回商談を行いました。
相手は購買部の鈴木部長です。

【制約】
- 200文字程度
- 丁寧な敬語、感謝の気持ちが伝わるように
- 次回アポイントの日程確認を含める
- 価格や見積もりの詳細には言及しない

【出力形式】
件名と本文を分けて出力してください。
```

## よくある出力形式の指定例

### 箇条書き
```
【出力形式】
箇条書きで5つ以内にまとめてください。
```

### 番号付きリスト
```
【出力形式】
優先順位の高い順に番号付きで列挙してください。
```

### 表形式
```
【出力形式】
以下の形式の表で出力してください：
| 項目 | 内容 | 担当 | 期限 |
```

### 見出し＋段落
```
【出力形式】
以下の構成で出力してください：
## 背景
## 課題
## 提案
## 期待効果
```

### JSON形式
```
【出力形式】
以下のJSON形式で出力してください：
{
  "title": "タイトル",
  "summary": "要約（100文字以内）",
  "keywords": ["キーワード1", "キーワード2", "キーワード3"]
}
```

---

## 困ったときは

- **結果が長すぎる** → 「〇〇文字以内で」と制約を追加
- **専門的すぎる** → 「初心者にもわかるように」と追加
- **的外れな回答** → 文脈情報（ペルソナ・ターゲット・状況）を詳しく書く
- **形式が違う** → 出力形式に「例」を書いて具体的に指定する
- **トーンが合わない** → 制約にトーンを明示（「カジュアルに」「フォーマルに」）
