# 講師ガイド

- プロジェクト名: AIポリテラシー育成プログラム Ver.2
- 対象: 講師（instructor）
- 作成日: 2026-01-05
- 版: v1.0

---

## はじめに

このガイドでは、講師として本プログラムを運営する際の操作方法を説明します。
講師は受講者の進捗を確認し、必要に応じて手動評価やフォローアップを行います。

## 4. 課題提出状況の確認

### 4.1 課題別提出状況

1. 左メニューから「課題管理」をクリック
2. 課題一覧が表示されます

**課題一覧の項目**:

| 項目 | 説明 |
|---|---|
| 課題名 | 課題タイトル |
| 種別 | 必須 / 任意 |
| 提出数 | 提出済み人数 / 全受講者数 |
| 平均スコア | 当該課題の平均点 |
| 評価待ち | 未評価の提出数 |

### 4.2 提出物の確認

課題名をクリックすると、当該課題の提出一覧が表示されます：

- 提出者名
- 提出日時
- スコア
- 評価状態（評価済み / 評価中 / 評価待ち / エラー）

さらに提出者名をクリックすると、提出内容と評価詳細を確認できます。

## 5. 手動評価の方法

### 5.1 手動評価が必要なケース

| ケース | 説明 |
|---|---|
| 自動評価エラー | Gemini APIのエラーで評価できなかった場合 |
| 再評価依頼 | 受講者から再評価の依頼があった場合 |
| 最終課題 | 最終発表資料など、人間の判断が必要な場合 |

### 5.2 手動評価の手順

1. ダッシュボードの「手動評価依頼」または課題管理から対象を選択
2. 提出内容を確認
3. 「手動評価」タブをクリック
4. 4軸それぞれにスコアを入力（各0-25点、合計100点満点）
5. フィードバックコメントを入力
6. 「評価を保存」をクリック

### 5.4 評価ガイドライン

評価時は以下のガイドラインを参考にしてください：

| スコア範囲 | 基準 |
|---|---|
| 90-100点 | 期待を超える成果物。独自の工夫や深い理解が見られる |
| 80-89点 | 要件を十分に満たしている。適切な取り組み |
| 60-79点 | 基本的な要件を満たしている。改善の余地あり |
| 40-59点 | 一部の要件のみ満たしている。追加学習が必要 |
| 0-39点 | 要件を満たしていない。再提出を強く推奨 |

## 6. レポート出力

### 6.1 利用可能なレポート

| レポート | 説明 | 形式 |
|---|---|---|
| 進捗サマリー | 全受講者の進捗一覧 | CSV / PDF |
| 視聴ログ | 動画視聴の詳細記録 | CSV |
| 提出履歴 | 課題提出の詳細記録 | CSV |
| 評価結果 | 評価スコアの一覧 | CSV / PDF |
| 週次レポート | 週ごとの活動サマリー | PDF |

### 6.2 レポート出力手順

1. 左メニューから「レポート」をクリック
2. 出力したいレポート種別を選択
3. 対象企業・期間を指定
4. 「レポート生成」をクリック
5. ダウンロードリンクが表示されます

**標準的なアジェンダ（60分）**:

| 時間 | 内容 | 担当 |

- [ ] 録画をアップロード（必要な場合）

## 10. よくある質問

### Q1: 自動評価の結果に違和感があります

**A**: 自動評価は参考値です。明らかに不適切な場合は、手動評価で上書きできます。
また、評価傾向は定期的に検証・調整しています。

### Q2: 受講者から再評価の依頼がありました

**A**: 提出内容を確認の上、手動評価で再評価を行ってください。
システム上は、手動評価が自動評価より優先されます。

### Q3: 特定の受講者のデータが見られません

**A**: 講師は全受講者のデータを閲覧可能です。
表示されない場合は、管理者に確認してください。
