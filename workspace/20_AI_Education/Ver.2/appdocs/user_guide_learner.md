# 受講者ガイド

- プロジェクト名: AIポリテラシー育成プログラム Ver.2
- 対象: 受講者（learner）
- 作成日: 2026-01-05
- 版: v1.0

---

## はじめに

このガイドでは、AIポリテラシー育成プログラム Ver.2 の受講方法を説明します。
13週間のプログラムを通じて、生成AIを業務で活用するスキルを身につけましょう。

---

## 1. ログイン・初期設定

### 1.1 初回ログイン

1. **招待メールを確認**
   - 件名: 「【AIポリテラシー育成プログラム】受講のご案内」
   - メールに記載された「アカウントを有効化する」ボタンをクリック

2. **パスワード設定**
   - 招待リンクをクリックすると、パスワード設定画面が表示されます
   - 8文字以上の安全なパスワードを設定してください
   - 大文字・小文字・数字を含めることを推奨します
   - ※ 招待リンクの有効期限は48時間です

3. **利用規約への同意**
   - 利用規約と個人情報の取り扱いを確認
   - 「同意する」をクリックして先へ進みます

4. **次回以降のログイン**
   - ログインURL: https://ai-literacy.example.com/login
   - 登録メールアドレスと設定したパスワードでログインします

### 1.2 プロフィール設定

1. 画面右上のアイコンをクリック → 「プロフィール」を選択
2. 以下の情報を確認・更新できます：
   - 表示名
   - メールアドレス（変更不可）
   - 通知設定

### 1.3 パスワードを忘れた場合

1. ログイン画面で「パスワードを忘れた方はこちら」をクリック
2. 登録メールアドレスを入力
3. 届いたメールのリンクから新しいパスワードを設定

## 3. 動画視聴の進め方

### 3.1 動画にアクセスする

1. 左メニューから「セッション一覧」をクリック
2. 視聴したいセッションを選択
3. 動画プレーヤーが表示されます

### 3.2 動画の構成

各セッションは2パートで構成されています：

| パート | 内容 | 時間 |
|---|---|---|
| Part 1 | 理論編（概念・考え方） | 約10分 |
| Part 2 | 実践編（演習・応用） | 約10分 |

### 3.3 視聴完了を記録する

1. 動画を最後まで視聴
2. 動画下の「視聴完了」ボタンをクリック
3. 確認ダイアログで「はい」を選択
4. 視聴完了が記録されます

> **重要**: 視聴完了ボタンを押さないと、進捗に反映されません。
> また、助成金申請に必要な証跡となりますので、必ず押してください。

### 3.4 視聴のコツ

- **集中できる環境で**: 動画は約20分/セッションです。まとまった時間を確保しましょう
- **メモを取る**: 重要なポイントや疑問点をメモしておくと、ミーティングで質問できます
- **繰り返し視聴OK**: 何度でも視聴できます。復習に活用してください

## 4. 課題提出の方法

### 4.1 課題一覧を確認

1. 左メニューから「課題一覧」をクリック
2. 課題の種類でフィルタリングできます：
   - 必須課題のみ表示
   - 任意課題のみ表示
   - 未提出のみ表示

### 4.2 課題の種類

| 種類 | 説明 | 修了要件 |
|---|---|---|
| **必須課題** | 修了に必要な課題（赤いバッジ表示） | 全12課題提出必須 |
| **任意課題** | 追加学習用の課題（青いバッジ表示） | 提出任意 |

### 4.3 課題に取り組む

1. 課題一覧から取り組みたい課題をクリック
2. 課題詳細ページが表示されます：
   - 課題の説明
   - 提出フォーマット
   - 評価基準（ルーブリック）

3. 課題内容を確認し、成果物を作成

### 4.4 提出方法

1. 課題詳細ページの「提出」セクションへスクロール
2. 提出形式を選択：
   - **テキスト入力**: 直接フォームに入力
   - **ファイルアップロード**: PDF、Word、画像などをアップロード
   - **URL入力**: GPTsリンクなど外部URLを入力

3. 内容を入力/アップロード
4. 「提出する」ボタンをクリック
5. 確認ダイアログで「はい」を選択

### 4.5 下書き保存

作業途中で保存したい場合：

1. 内容を入力した状態で「下書き保存」ボタンをクリック
2. 次回アクセス時に続きから作業できます

> **注意**: 下書きは提出ではありません。必ず「提出する」ボタンで提出してください。

### 4.6 提出後の流れ

1. **自動評価**: 提出後、AIが自動で評価を行います（通常5〜10分）
2. **通知**: 評価完了時にメールでお知らせします
3. **結果確認**: ダッシュボードまたは課題詳細ページで結果を確認

### 5.3 スコアの目安

| スコア | 評価 | 説明 |
|---|---|---|
| 90-100 | 優秀 | 期待を超える成果物 |
| 80-89 | 良好 | 要件を十分に満たしている |
| 60-79 | 合格 | 基本的な要件を満たしている |
| 0-59 | 要改善 | 再提出を推奨 |

### 5.4 再提出について

- スコアに関わらず、再提出が可能です
- より良い成果物を目指して、フィードバックを参考に改善してみましょう
- 再提出した場合、最新の提出が評価対象となります

## 6. ミーティングへの参加

### 6.1 ミーティング概要

| 項目 | 内容 |
|---|---|
| 頻度 | 隔週（全8回） |
| 時間 | 60分 |
| 形式 | Zoom（オンライン） |
| 内容 | Q&A、グループディスカッション、講師からの講評 |

### 6.2 参加方法

1. ダッシュボードの「次回ミーティング」セクションを確認
2. 開始時刻にZoomリンクをクリック
3. Zoomが起動し、ミーティングに参加

> **Zoomリンク**: ダッシュボードまたはセッションページに掲載しています

### 6.3 ミーティングでの準備

- 事前に該当週の動画を視聴しておく
- 質問したいことをメモしておく
- マイク・カメラが使える環境を準備

### 6.4 欠席する場合

- 事前連絡は不要ですが、録画は後日共有されます
- 修了要件にミーティング参加は含まれませんが、学習効果を高めるため参加を推奨します

## 7. 最終発表について

### 7.1 最終発表の概要

Week 13 に行う最終発表は、プログラムの集大成です。

| 項目 | 内容 |
|---|---|
| 時間 | 20分（発表15分 + 質疑5分） |
| 形式 | 画面共有でスライド発表 |
| 内容 | 作成したGPTsの紹介と運用結果 |

### 7.2 発表構成（推奨）

1. **対象業務と課題**（3分）
   - どんな業務に取り組んだか
   - 何が課題だったか

2. **GPTs設計・実装**（5分）
   - 設計プロセス（メタプロンプト活用）
   - Instructions・知識ベースの概要
   - 工夫した点

3. **実運用の結果**（5分）
   - 誰が、どう使ったか
   - 実際の対話ログ（抜粋）
   - 得られた効果・課題

4. **振り返りと今後**（2分）
   - 学んだこと
   - 今後の改善・展開計画

### 7.3 発表資料の提出

- 発表資料（スライド）は EX-26 として提出してください
- 提出期限: 最終発表会の前日まで

## 8. よくある質問（FAQ）

### Q1: 動画が再生できません

**A**: 以下を確認してください：
1. インターネット接続を確認
2. ブラウザを最新版に更新
3. 別のブラウザ（Chrome推奨）で試す
4. 解決しない場合はサポートへ連絡

### Q2: 課題の提出期限はありますか？

**A**: 厳密な期限はありませんが、各週の課題は該当週のミーティングまでに提出することを推奨します。
最終的には、Week 13 の最終発表までにすべての必須課題を提出してください。

### Q3: 評価スコアが低かったのですが、修了できますか？

**A**: はい、修了できます。修了要件は「提出すること」であり、スコアは要件に含まれません。
ただし、フィードバックを参考に再提出して、スキルを高めることを推奨します。

### Q4: ChatGPT Plus/Team のアカウントがありません

**A**: Phase 3以降の課題ではGPTsを使用します。開始までにご準備ください。
費用については企業負担または自己負担となります。詳細は講師にお問い合わせください。

### Q5: ミーティングに参加できない場合はどうすればいいですか？

**A**: 事前連絡は不要です。録画は後日共有されますので、そちらをご覧ください。

### Q6: 他の受講者の提出物を見ることはできますか？

**A**: プライバシー保護のため、他の受講者の提出物は閲覧できません。
ミーティングでのグループディスカッションで、お互いの取り組みを共有する機会があります。

### Q7: 修了証はどのように受け取れますか？

**A**: 修了要件を達成すると、メールで修了証（PDF）が送付されます。
ダッシュボードからもダウンロード可能です。
