## 作業手順テンプレート

### Step 1: 演習位置の特定
```bash
grep -n "^## 演習[0-9]" <ファイル名>.md
```

### Step 2: ファイル読み込み
```
Read tool で該当セクション(offset, limit指定)を読み込み
```

### Step 3: 修正実行
```
Edit tool で以下の形式に修正:
- 演習タイトル: `## 演習N: <タスク名>（XX分）`
- ステップごとに所要時間を括弧で追加
```

### 所要時間の目安
- **情報整理系**: 10-20分
- **作成・記述系**: 15-25分
- **テスト・検証系**: 20-30分
- **レビュー・ログ系**: 10-15分
