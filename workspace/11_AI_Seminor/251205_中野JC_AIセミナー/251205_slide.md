# Part 1
## 生成AIの基礎とリスク

---

## 生成AIとは？

- **AI（人工知能）:** データから学習するプログラム
- **生成AI:** 学んだデータをもとに**新しいものを創り出す**AI
  - テキスト、画像、音楽、プログラムコードなど

---

## 主要な生成AIツール

| ツール | 特徴 | 料金 |
| :--- | :--- | :--- |
| **ChatGPT** | 対話・文章作成に強い | 無料～$20/月 |
| **Gemini** | Google連携・最新情報 | 無料～¥2,900/月 |
| **Claude** | 長文処理・分析に強い | 無料～$20/月 |
| **Copilot** | Microsoft連携 | 無料～$30/月 |

---

## なぜ今日はGeminiを使うのか？

1. **Google連携**
   - NotebookLMと同じGoogleアカウントで使える

2. **無料でも高機能**
   - 今日のワークは無料版で全て可能

3. **使いやすさ**
   - シンプルなインターフェース

---

## 【重要】リスクと対策

| リスク | 対策 |
| :--- | :--- |
| **情報漏洩** | 機密情報・個人情報は**絶対に入力しない** |
| **誤情報** | 出力は必ず**ファクトチェック**する |
| **著作権** | 商用利用前に**利用規約を確認** |

**今日のワーク:** 実名・具体的数値は避けてください

# Part 2
## プロンプトの基本

---

## プロンプトとは？

AIへの「指示書」のこと

**良いプロンプト = 良いアウトプット**

---

## プロンプトの「型」

```
【役割】あなたは〇〇の専門家です

【背景・目的】〇〇を実現したい

【制約】予算・期間・対象

【出力形式】箇条書き・表・段階的に
```

---

## 悪いプロンプトの例

```
事業計画書を作って
```

**問題点:**
- 何の事業か不明
- どんな形式で欲しいか不明
- 誰向けか不明

---

## 良いプロンプトの例

```
あなたは経営コンサルタントです。

私は飲食店を経営しており、新規事業として
テイクアウト専門店を立ち上げたいと考えています。

以下の形式で事業計画書の構成案を作成してください。
- 事業概要
- 市場分析
- ターゲット顧客
- 収支計画
- リスクと対策
```

## AIが得意なこと・苦手なこと

### 得意なこと
- 文章の要約・整理
- アイデア出し・ブレインストーミング

## プロンプトのコツ：追加指示と改善ループ

最初の出力が物足りない場合は、
**追加指示**で改善:

```
「もっと具体的に」
「若い世代向けのトーンに変えて」
「箇条書きを表形式に変換して」
「デメリットも含めて分析して」
「反対意見も出してみて」
```

**プロンプト4の改善ループ活用例:**
→ AIが提案した「強化すべきポイント」を選ぶだけ
→ 「ターゲット顧客をもっと詳しく」と指示
→ 精度が上がった計画書が再生成される
