# SUNZグループ向け生成AIセミナー 企画書

## 1. プロジェクト概要

- **セミナー名:** AI活用による事業計画策定セミナー ～2030年10億円達成への道筋を描く～
- **開催日:** 2025年12月26日（金）13:00〜15:00
- **対象:** SUNZグループ幹部8名
- **時間:** 120分（セミナー25分＋ワークショップ75分＋発表・質疑20分）
- **会場:** SUNZグループ本社（豊岡）
- **講師:** 井戸聞多（合同会社HaRu 代表社員）
- **サポート:** 現地サポート1名（操作支援）
- **予算:** 20万円（税別、交通費別）

---

## 2. SUNZグループの背景と課題

### 企業概要
- 豊岡に本社を置く多角化企業グループ（元パチンコ事業）
- 事業領域：パチンコ、介護施設、飲食店、ホテル・民泊、ヘッドスパ、メガネ等
- 2030年目標：10億円の利益達成

### 主要課題
1. **事業計画の必要性:** 各事業責任者が2030年に向けた具体的な事業計画を策定する必要がある
2. **AI活用の遅れ:** 特に財務部門は未だ紙ベースで運用、DX推進が急務
3. **部門間連携不足:** 各部門が自立しており、相互理解・協業が不十分
4. **戦略的思考の醸成:** 現状把握から将来ビジョンへの道筋を描く力の強化が必要

---

## 3. セミナーの戦略的目標

### ミッション
参加者が生成AIを活用して、わずか2時間で実用的な事業計画書のたたき台を作成できることを体験し、その後の3時間の会議で本格的な事業計画へとブラッシュアップするための基礎スキルを身につけること。

### 大目的（要件定義で確定）
**「2030年に向けて自分がやるべきことが明確になった」という腹落ち**

### サブ目的
- 苦手だった事業計画作成が、生成AIをパートナーにすることで「できる」という自信を得る
- 体験の質で継続サポートへの関心を自然に喚起する（セミナー中の言及はしない）

### 具体的目標
1. **即戦力化:** セミナー終了後、すぐに続く会議で活用できる計画書を作成
2. **スキル習得:** AIプロンプトの基本を理解し、自走できる能力を獲得
3. **意識改革:** AI = 難しい技術ではなく、思考を整理し加速させるパートナーという認識
4. **相互理解:** 各部門の事業計画発表を通じて、他部門への理解を深める

### 参加者のAIレベル
- **ほぼゼロ**（AIツールの利用経験がほとんどない前提で設計）
- 午前中から会議を実施しているため、ラポール形成は最小限でOK

### 5.2. セミナー（20分）13:05〜13:25

#### 構成

| 時間 | 内容 | ポイント |
| :---: | :--- | :--- |
| 3分 | 生成AIとは・主要ツール | Geminiを選ぶ理由、リスクと対策 |
| 5分 | プロンプトの基本 | 良いプロンプトの「型」を理解 |
| 12分 | 事業計画作成デモ | 介護事業を例に4ステップ実演 |

#### デモの流れ（介護事業を例に）

**使用する4つのプロンプト:**
1. **現状分析プロンプト** → SWOT分析を自動生成
2. **市場環境プロンプト** → 業界動向・競合分析
3. **戦略立案プロンプト** → 2030年への成長戦略
4. **計画書完成プロンプト** → 全体をまとめて計画書化

**【対話型プロンプトの特徴】（要件定義で確定）:**
全プロンプトで、AIは以下の3点をセットで回答する：
1. **その時点のまとめ**（現状の整理）
2. **より良くするアイデア**（3つ程度）
3. **深掘りする質問**（次に考えるべきこと）

→ 対話を重ねるほど計画が良くなる体験を設計

**ポイント:**
- 各プロンプトは配布資料に記載、コピペで使える
- 【】内を自社情報に書き換えるだけ
- 段階的に思考が深まる様子を実演

### 5.3. ワークショップ前半（35分）13:25〜14:00

#### Phase 1: 事業選択と現状分析（15分）

**参加者の想定事業テーマ:**
- 三山社長：今帰仁そば全国展開
- 小西常務：保育事業の採用戦略・規模拡大
- 杉本部長：ハラルラーメン事業拡大
- 有村部長：730ホテル再生計画
- 長峯部長：介護施設展開戦略
- 山口部長：パチンコ事業の収益最適化
- マイコCHO：新規事業探索
- 野原経理部長：DX推進計画

**作業内容:**
1. 扱う事業テーマを決定（上記は例、変更可）
2. プロンプト1で現状分析（SWOT）
3. プロンプト2で市場環境分析

**【演出】最初の1回を全員同時に（採用）:**
- 最初のプロンプト（SWOT分析）は全員同時に送信
- 「では、皆さん準備はいいですか？3, 2, 1...送信！」
- 一人で始める不安を解消し、一体感を生む

**【挙手確認】サポート型（採用）:**
- 各フェーズ終了ごとに「困っている方いますか？」と確認
- 全4回実施（SWOT完成後、市場分析後、戦略立案後、計画書完成後）

#### Phase 2: 戦略立案（20分）

**作業内容:**
1. プロンプト3で2030年ビジョンと戦略を作成
2. 必要に応じて追加質問で深掘り
3. 数値目標の設定

### 5.4. ワークショップ後半（40分）14:00〜14:40

**【橋渡し】8名サマリーの即興作成（採用）:**

**注：** 2時間という時間制約上、個別のスライド作成は省略し、インフォグラフィックで代替

## 10. リスク管理

### 技術的リスク
| リスク | 対策 |
| :--- | :--- |
| ネット接続不良 | モバイルWi-Fi持参、テザリング案内 |
| Gemini応答遅延 | 時間帯をずらす、代替プロンプト用意 |
| PC操作に不慣れ | 現地サポート1名が個別対応、挙手確認で早期発見 |

### 参加者のAIレベル対応（ほぼゼロ前提）
| 懸念シナリオ | 対策 |
| :--- | :--- |
| 「操作がわからない」で止まる | 現地サポートが巡回、挙手確認を4回実施 |
| 「AIの回答が期待と違う」で困惑 | マジックワード集を配布、追加指示の仕方をデモで見せる |
| 「自分には無理」と心理的に離脱 | 最初の1回を全員同時送信、達成感を共有 |

### 内容的リスク
| リスク | 対策 |
| :--- | :--- |
| 2時間で完成しない | 「たたき台」であることを強調 |
| 機密情報の扱い | 冒頭で注意喚起、匿名化を推奨 |
| 期待値のズレ | AIの限界も正直に伝える |
