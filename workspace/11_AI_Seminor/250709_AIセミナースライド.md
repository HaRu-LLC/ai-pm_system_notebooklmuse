# 思考を深めるプロンプトの「型」
1. **役割を与える:**
   - 「あなたは優秀な経営コンサルタントです」
2. **背景を伝える:**
   - 「私の会社は...という課題を抱えています」
3. **深掘りを命じる:**
   - 「そのアイデアのメリット・デメリットは？」
4. **多角的な視点を要求する:**
   - 「財務、マーケティング、人事の観点からアドバイスをください」

# 【例】スライド生成プロンプト
以下の内容で、プレゼン資料の構成案を作成してください。

- **テーマ:** 競合が気づいていない「〇〇」という新顧客層へのアプローチ
- **目的:** 経営会議で、この新市場への参入許可を得る
- **ターゲット:** 弊社の経営陣
- **含める要素:**
    1. ターゲット市場の規模と可能性
    2. 具体的なアプローチ手法
    3. 必要な予算と予測ROI
    4. リスクと対策
- **トーン:** 論理的かつ情熱的に
- **スライド枚数:** 全8枚
- **発表時間:** 10分
- **形式:** Marp形式のマークダウンで記述
