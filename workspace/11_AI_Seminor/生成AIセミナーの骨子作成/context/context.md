# Context

- Project: 生成AIセミナーの骨子作成（中野JC向け）
- Goal: 90分（講義30分＋ワーク30分×2）で受講者が自作GPTのプロトタイプを構想できるセミナー骨子と付随資料を整備する
- Key Outputs:
  - 要件整理 `intake/answers.md`, `intake/summary.md`
  - 実行計画 `plan/plan.md`
  - 選定エージェント `plan/selected_agents.json`（Planner/Communicator/Example/Critic）
  - セミナー骨子 `outputs/final_answer.md`
  - 意思決定ログ `runs/decision_log.md`
  - 追加資料: ワークシートテンプレート/記入例 (`artifacts/workshop_sheet_*.md`), デモ準備リスト・サンプル (`artifacts/demo_materials_checklist.md`, `artifacts/gpt_demo_samples.md`)
  - 成果物レポート `outputs/report.md` とナレッジ `knowledge/20251011_201234_生成AIセミナーの骨子作成.md`
- Notes: 講義は「導入→仕組み→自作ステップ→運用ポイント」で構成、ワークは課題整理とプロト設計の二段階。特産品販促・観光案内・経理補助をデモ例として準備。
